## Uruchomienie lokalne

Pobierz repozytorium i otwórz `index.html` w przeglądarce — żadnych instalacji, serwera ani połączenia z internetem nie wymaga.

```bash
git clone https://github.com/jacksuliga-rgb/darmowenarzedzia.git
cd darmowenarzedzia
# otwórz index.html w przeglądarce
```

## GitHub Pages

1. Wejdź w **Settings → Pages**
2. Source: `Deploy from a branch`
3. Branch: `main` / folder: `/ (root)`
4. Zapisz — strona będzie dostępna pod `https://jacksuliga-rgb.github.io/darmowenarzedzia`
